# Responsive Layout Patterns: The Layout Lab

A curated collection of modern, responsive grid and flexbox experiments designed for high-performance UI/UX. This library serves as a "Layout Lab" where design principles meet technical implementation.

## 🚀 Overview
As a graphic design graduate transitioning into web development, I created this repository to bridge the gap between static design systems and flexible, responsive code. Each pattern focuses on clean, modular CSS using a strict **camelCase** naming convention.

## ✨ Key Features
* **Unique Grid Systems:** Non-traditional layouts including Bento grids and masonry styles.
* **Flexbox Experiments:** Deep dives into fluid, multi-direction alignment.
* **Responsive First:** All patterns are tested across various screen breakpoints.
* **CamelCase Architecture:** Clean, consistent class names for professional integration (e.g., `.mainGridContainer`).

## 🛠️ Tech Stack
* **Vite + React:** Modern build tooling for fast development.
* **Tailwind CSS:** For rapid utility-based styling.
* **React Router:** Integrated for multi-page pattern navigation.
